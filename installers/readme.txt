This directory should contain zip distribution of JRE, MySQL and Tomcat.
The JRE should be a named jre.zip and should contain a directory named "jre".
The MySQL and Tomcat could be of any name but the zip filename
and the expanded directory name should be in the build.xml file's:
	tomcat.zip
	tomcat.zip.expanded
	mysql.zip
	mysql.zip.expanded
properties.
